Is this ready for GitHub Pages or Netlify?

Yes. It is a static HTML, CSS, and JavaScript app with no build step.

Where is authentication stored?

For the frontend demo, users and sessions are stored in localStorage. The code is organized for a future REST API migration.

Does it include backend architecture?

The README includes FastAPI, SQLite, and REST API migration guidance for the next phase.
