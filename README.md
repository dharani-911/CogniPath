# CogniPath – AI-Powered Career Intelligence Platform

## Overview
CogniPath is an AI-powered career guidance platform that helps students and job seekers discover suitable career paths through personalized recommendations, skill gap analysis, and learning resources.

## Features
- User Registration and Login
- Secure Authentication
- Career Guidance Dashboard
- Skill Gap Analysis
- Personalized Recommendations
- SQLite Database Integration
- FastAPI Backend
- Responsive Frontend

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI (Python)
- Database: SQLite
- Version Control: Git & GitHub

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/CogniPath.git
   ```

2. Navigate to the project folder.

3. Install dependencies:
   ```bash
   pip install -r backend/requirements.txt
   ```

4. Start the backend:
   ```bash
   uvicorn main:app --reload
   ```

5. Open the frontend in your browser.
